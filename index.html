Preface

This is my ICS4U portfolio, where I organized everything I learned during the two and a half quadmester into a single website. To describe how this learning process went, I can confidently say that I have been able to develop better learning skills in regards to computing in the way we were given the course. To have the freedom to learn and process all the new content at your own pace while also being given the opportunity to learn other concepts has enabled me to learn much more than expected. It also gave me more tools on how to code outside of the classroom, making the newly-formed skills useful in adjacent to the class and to my personal enjoyment. I also was able to use the new skills I learned to create stuff that was useful to me in a way unrelated to the class. The reason why I consider these skills to be useful is because I have intentions on pursuing Computer Science after high school, so the more skills I can pick up will be useful to me.

Computer Science Principles: Programming

This mandatory programming module helped to be able to critically understand and process code in the form of pseudocode in order to visualize what the code is asking. The examples of what you were tested on was mainly algorithm questions that truly required you to think in the sense that it wasn't a traditional language, but simply more of them giving you concept questions to see if you understand. By that logic, it was a pretty valuable start to the course since it enabled you to think effectively in regards to coding so that you could bring this mindset over to the real language concepts in regards to web developement. I achieved complete mastery on this module on Khan Academy.

Specfic examples of the units I was tested on:

  • Variables
  • Mathematical expressions, procedures and constants
  • String manipulation (concatation, etc)
  • Conditional statements including nested conditionals and booleans
  • Functions and how to call and define them
  • Loops and iterating through them

Computer Science Principles: Digital Information

Although this wasn't officially an assigned module, I decided to go out of my way to benefit myself in terms of what I understand about computing principles. So, I learn and tested on what digital information is and how it essentially built the foundation for computers today. The most important thing I learned from this was understanding how binary works and how to decode binary to numbers using doubling numbers somewhat in the sense of what a factorial is with 1,2,4,8 and so forth. It also showed me what analog data, data compression and digital licenses are which weren't as insightful as binary concepts but were still important to learn. So now I also understand what licenses such as the Creative Commons on. This helped me as a student since I now know how the basis of all computing works which can potentially help me out in the future when I pursue more education in this field. I achieve full mastery on this course in Khan Academy.

Intro to JavaScript: Drawing & Animation

This was the first coding based module I worked on in the semester. I found the majority of this to be review since the syntax and concepts were identical to what I learned in ICS3U with C++. The one thing I newly learned was object oriented programming, which I didn't know was a thing yet. I don't really see a use for objects in our level of coding however I see the potential it will have for projects in the years to come, so to be familiar with it now will be beneficial. Apart from that, the code was being presented with Processing JS, a library to make visualizations which helped a lot in demonstrating the code. I would assume if this was my first time ever coding, the shapes and animations made would've helped a lot on seeing the code of what it presents and not simply being a bunch of text that makes commands. This will help me as a student in the further since it these skills will carry on (and already has) in terms of what I am developping in front-end projects.

Intro to HTML/CSS: Making webpages

Now before I actually took the course, every time I saw HTML whether it be through Inspect Element or in various medias I was always very confused. To me, it looked harder to write than actual code such as the C++ or ActionScript I learned in previous classes due to the way everything was organized. Now that I finally finished the course and learned how to make websites (with this site entirely being HTML), I can conclude that it is much easier than real coding than what I initially thought. Now the interesting part of HTML is that since it is the foundation for a majority of websites, it is impressive to see how people organize it with the addition of CSS to make visually stunning results. Those skills helped me learn how to make headers, text, paragraphs, markdowns and much more, and then with the addition of CSS I was able to learn a lot more. I find CSS interesting because it has the ability to make a boring simple website into something like an interactive site that can even have animations. HTML has the potential to be something I do in my future , depening on if I want to go more in depth of Web Developement, so now that I have the basics down my background in designing websites can be beneficial in the future. The biggest impact on this course as a student was for sure how creative you can get with websites.
